# FlappyHorst
Ein simples, auf Flappy Bird basierendes Spiel, welches dem Charakter "Horst Flock" gewidmet ist.

## Steuerung
SPACE | Linke Maustaste | Rechte Maustaste - Springen

ESCAPE - Pause

F3 - Debug

? - Easteregg :D
